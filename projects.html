building a home media server

always use tor or a vpn!!

(note: there are countless approaches to this. search the web to find what works best for your means and needs!)

prerequisites:
-a raspberry pi 3b+ that runs a terminal

music download your spotify music using spotify_sync.
if something isnt on spotify, get it using soulseek or find it elsewhere (e.g bandcamp).
movies & shows torrent your movies & shows using aria2.
there are sites like torlock that help you find the magnet link needed for torrenting.
podcasts download your podcasts using castget.
there are sites like getrssfeed that help you find the rss feed link needed for downloading.
books & zines download your books & zines using wget.
you can use archive.org to find the epub link needed for downloading.
for lesser known things / zines you will likely need to find sites specific to your interest, e.g theanarchistlibrary.
radio wip. will likely do it using lyrion music server.
newspaper you dont even need a server for this. just use a paywall remover
